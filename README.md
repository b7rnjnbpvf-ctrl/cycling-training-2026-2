# AI 英语生词学习助手

手机端优先的在线英文学习网站第一版。

## 当前功能

- 首页极简：翻译输入框 + 生词输入框
- 中英翻译 mock
- 单词音标、释义、发音按钮
- 生词本自动记录
- 历史词库永久保存
- 测试通过后从当前生词本移除，历史仍保留
- 口语练习 mock
- 间隔重复复习计划
- 数据保存在浏览器 localStorage
